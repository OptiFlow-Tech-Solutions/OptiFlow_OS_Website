# /benchmark
Measure performance across all pages.

## Metrics
1. Page weight (HTML + CSS + JS + images)
2. Request count per page
3. First Contentful Paint estimate
4. Largest Contentful Paint estimate
5. CSS complexity (selector count, specificity depth)
6. JS execution time estimate
7. Performance budget compliance

## Agent
Uses benchmark skill for measurement, benchmark-optimization-loop for improvements.
