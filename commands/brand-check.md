# /brand-check
Audit all pages for brand compliance.

## Checks
1. Color ratio (88% neutral / 12% brand) per page
2. Voice compliance (use/avoid vocabulary from DESIGN.md)
3. Typography consistency (Inter usage, type scale adherence)
4. Component consistency (buttons, cards, spacing)
5. Dark mode coverage audit
6. Accessibility score per page

## Agent
Uses brand-voice for copy audit, frontend-design-direction for visual audit, accessibility for a11y.
