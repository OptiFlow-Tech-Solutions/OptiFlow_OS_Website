# SEO-003: Social Media OG Image (1200x630)
- **Status:** Pending | **Priority:** LOW | **Est. Hours:** 0.5
- **Category:** SEO | **Phase:** 09
- **Score Impact:** SEO +4

## Problem / Expected / Affected
`og:image` currently points to the square logo PNG (512x512), which gets poorly cropped or rejected by social platforms expecting a 1200x630 card. Shares on Twitter, LinkedIn, and Facebook display a clipped or absent preview image, reducing click-through from social channels.

Expected: A purpose-built 1200x630 PNG social card with branding, a short tagline, and a soft CTA. Both `og:image` and `twitter:image` meta tags should reference this new asset.

## Acceptance Criteria
- [ ] 1200x630 PNG social card asset created in `assets/img/`
- [ ] `og:image` meta tag on every page points to the new asset
- [ ] `twitter:image` meta tag on every page points to the new asset
- [ ] Image includes recognizable OptiFlow OS branding, a tagline, and a CTA element

## Implementation
Design the social card image and place it in `assets/img/`. Update the OG meta injection in `assemble.mjs` to reference the new asset path instead of the existing logo.

## Dependencies / Definition of Done
- **Depends on:** Design asset creation (image file)
- **DoD:** Facebook Sharing Debugger and Twitter Card Validator both render the card correctly.
