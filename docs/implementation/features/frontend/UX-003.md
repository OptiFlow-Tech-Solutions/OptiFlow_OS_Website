# UX-003: Dark Mode Logo in Navigation
- **Status:** Pending | **Priority:** LOW | **Est. Hours:** 0.2
- **Category:** Design System | **Phase:** 06
- **Score Impact:** Frontend +1

## Problem
Footer logo applies filter:brightness(10) in dark mode, but the navigation logo does not, making it invisible on dark backgrounds.

## Expected
Navigation logo is visible in dark mode without being over-brightened in light mode.

## Affected
- **Pages:** All
- **Files:** core.css

## Acceptance Criteria
- [ ] Navigation logo visible against dark backgrounds in dark mode
- [ ] Navigation logo appearance unchanged in light mode

## Implementation
Add `[data-theme="dark"] .topnav .logo-img { filter: brightness(10); }` to core.css to match the existing footer logo behavior.

## Dependencies
None
