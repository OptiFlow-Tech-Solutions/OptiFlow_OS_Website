# CNT-004: Favicon Variants
- **Status:** Pending | **Priority:** LOW | **Est. Hours:** 0.3
- **Category:** Content | **Phase:** 07
- **Score Impact:** Frontend +1

## Problem
Only a PNG favicon exists; no .ico or SVG variant for broader browser and platform support.

## Expected
favicon.ico, PNG favicon, and SVG favicon generated, with build pipeline handling all variants.

## Affected
- **Pages:** All
- **Files:** assemble.mjs, favicon files

## Acceptance Criteria
- [ ] favicon.ico generated and served
- [ ] SVG favicon linked for modern browsers
- [ ] Build pipeline handles favicon generation/injection
- [ ] All variants present in dist/ after build

## Implementation
Add a favicon generation step to assemble.mjs that converts logo.png to favicon.ico. Add an SVG favicon link tag to the page head assembly. The build pipeline injects the appropriate <link> tags for each format.

## Dependencies
None
