# UX-005: Breadcrumb Navigation
- **Status:** Pending | **Priority:** LOW | **Est. Hours:** 1.5
- **Category:** Navigation | **Phase:** 06
- **Score Impact:** Frontend +3, UI/UX +2

## Problem
CSS placeholder for .breadcrumb exists in core.css but the component was never built.

## Expected
Auto-generated breadcrumb navigation on all non-home pages with BreadcrumbList structured data.

## Affected
- **Pages:** All 15 non-home pages
- **Files:** core.css, assemble.mjs

## Acceptance Criteria
- [ ] Breadcrumbs visible on all non-home pages
- [ ] Breadcrumb trail auto-generated from URL path segments
- [ ] BreadcrumbList JSON-LD structured data injected
- [ ] All breadcrumb segments are clickable links
- [ ] Existing .breadcrumb CSS from core.css is used

## Implementation
Add breadcrumb generation logic to assemble.mjs that parses the page's URL path, capitalizes and linkifies each segment, and injects the markup and JSON-LD structured data. Uses the existing .breadcrumb CSS class already defined in core.css. No new partial file needed — the builder generates it per page.

## Dependencies
UX-004 (page styles must be extracted first to avoid conflicts)

## Definition of Done
- [ ] Code implemented
- [ ] `npm run validate` passes
- [ ] `npm test` passes
